## âš™ï¸ How It Works

1. **Select a Section**  
   The user chooses a specific AFOQT section from the home screen.

2. **Dynamic Question Loading**  
   The application loads the corresponding JSON question bank in real time.

3. **Section-Based Timer Starts**  
   A countdown timer begins based on official AFOQT time limits for that section.

4. **One Question at a Time Interface**  
   Questions are displayed individually to simulate real exam conditions.

5. **Answer Tracking in Memory**  
   User selections are stored instantly in browser memory (no backend required).

6. **Automatic Completion Handling**  
   When the user finishes or time expires:
   - Score is calculated
   - Percentage is generated
   - Correct vs selected answers are displayed
   - Incorrect answers are clearly highlighted
   - Performance-based readiness feedback is shown
